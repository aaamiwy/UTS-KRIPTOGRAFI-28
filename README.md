# UTS-KRIPTOGRAFI-28
Symmetric Cryptography dan Asymmetric Cryptography

## Deskripsi

Program ini dibuat untuk mengimplementasikan dan membandingkan kriptografi simetris menggunakan Fernet dengan kriptografi asimetris menggunakan RSA-2048.

## File Program

### 1. SimetrisFernet.py

Implementasi kriptografi simetris menggunakan Fernet untuk proses enkripsi dan dekripsi data.

### 2. AsimetrisRSA.py

Implementasi kriptografi asimetris menggunakan RSA-2048 untuk proses enkripsi dan dekripsi data.

## Fitur

* Input plaintext dari pengguna
* Enkripsi data
* Dekripsi data
* Pengukuran waktu enkripsi
* Pengukuran waktu dekripsi
* Perhitungan ukuran ciphertext
* Perbandingan karakteristik Fernet dan RSA

## Cara Menjalankan

Install library:

```bash
pip install cryptography
```

Jalankan program Fernet:

```bash
python SimetrisFernet.py
```

Jalankan program RSA:

```bash
python AsimetrisRSA.py
```
